HTML

  • The head element contains information about the webpage.
  • The head element is where to define a linked css file that holds information applied across the webpage
  • The body element represents the visible content shown to the user.
  • At the end of the boody element is where to define the link to a js file that adds functionality to the webpage

CSS

  • Most common format of css coding is 'external css style sheet' which is a seperate file programmers link to their html file
  • CSS code typically follows this rule: "selector {declaration}
  • Declarations follow the format "property:value"
  • An example of a property is 'margin' which indicates how much space we want around the outside of an element.
  • Another example of a property is padding which indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git branch: lists all branches of the repository
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes to the local branch
  • git commit -m "Message about commit": commits the local branch to the github branch with the user message
  • git pull origin main: pulls the current updated main branch from github
  • git push origin "branch_name": pushes branch to github and makes pull request available

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if() statement will only exicute code if the notion in the parentheses is TRUE
  • Programmer can also define else if () that checks another notion if the no previous notions are true
  • And lastly the else () statement will execute if all previous notions are false
  • A function is a user defined set of instructions that can be called upon throughout the program
  • console.log("message") prints "message" or any value/variable to the console log
  • console log is hidden from view but if user "inspects" the webpage (command + option + i) it is viewable
